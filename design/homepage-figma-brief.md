# ClimateFood Intelligence — Homepage Figma Brief

This file is a Figma-ready layout brief that can be imported manually into a Figma project:

- Frame: Desktop 1440x3000
- Sections: Sticky Navbar, Hero Slider, Dashboard Preview Cards, Map Section, Predictions Section, Footer
- Style: SaaS, modern glassmorphism, emerald/blue accents
- Components:
  - Navbar with links: Home, Dashboard, Climate Data, Food Supply, AI Predictions
  - Hero with headline + CTA + slider pagination dots
  - 3 analytics cards for climate/air/crops
  - Full-width map panel
  - Prediction callout card
  - Theme toggle (dark, light, gradient)

Typography:
- Inter Bold 64 (hero)
- Inter Semibold 24 (section titles)
- Inter Regular 16 (body)

Color tokens:
- bg-dark: #020617
- bg-light: #F8FAFC
- primary: #22C55E
- accent: #3B82F6
- text-muted: #94A3B8
